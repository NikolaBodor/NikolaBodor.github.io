AGES Learning Model

The AGES learning model has four components, attention, generation, emotion and spacing.

  • The attention component suggests that to properly learn you need to grab and hold the attention of whoever is learning.
  • The generation component suggests that to help with learning you need to actively connect new concepts with what you already know to solidfy you understanding of the new concepts.
  • The emotion component suggests that emotions can significantly influence memory. Positive emotional states can boost your memory and learning while negative emotional states can delay and set you back.
  • The spacing component suggests that learning isn't a one time event and that it requires multiple sessions over time for the information to become permanent.
By understanding these four component, poeple can put them into practice and learn more effectively.

Ways to increase your neuroplasticity

  • Learning a new skill e.g. a language or an instrument
  • Watch a documenty series
  • Read a book
  • Try do a puzzle of some sort e.g. jigsaw
